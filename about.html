Tentang Proyek Saya

Website ini merupakan proyek Desain Web yang saya kerjakan sebagai tugas membuat simulasi website dasar. Tujuan dari proyek ini adalah mengaplikasikan struktur HTML dan styling CSS untuk membuat website yang rapi dan fungsional.

Melalui proyek ini, saya belajar bagaimana menghubungkan halaman-halaman web, membuat navigasi yang interaktif, dan menerapkan desain yang konsisten dengan tema estetika pastel yang saya sukai.

Keahlian yang Dipelajari:

  • Struktur HTML yang rapi dan terorganisir
  • Styling dengan CSS: warna, font, dan layout
  • Membuat navigasi antar halaman
  • Menggunakan elemen-elemen form
